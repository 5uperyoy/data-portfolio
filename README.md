# Data Portfolio – Jorge García

Este repositorio contiene proyectos de análisis de datos enfocados en:
- Limpieza y exploración de datos con Python
- Análisis con SQL
- Visualización de datos con Power BI

## Estructura del repositorio
- /python → Notebooks y scripts de análisis
- /sql → Consultas SQL
- /powerbi → Dashboards y visualizaciones
- /datasets → Conjuntos de datos utilizados

## Objetivo
Desarrollar habilidades prácticas para un puesto junior de Data Analyst.
